  • 个人开源框架/库:

    • JavaScript & Node.js
      • RocketEngine: npm模块. 一个完整的超轻量级Node.js WebSocket库, 无第三方模块依赖,使用简单,功能丰富.
      • wsframe: 从RocketEngine中分离出的WebSocket Frame解析器/生成器
      • execq: npm模块. 用于RocketEngine库内部, 可以阻塞当前任务的,并在合适时候执行他们, 目的是处理异步工作流.
      • event.js: 用于RocketEngine前端库内部, 一个在浏览器端实现异步操作的事件驱动编程框架,依照Pub/Sub观察者体系模式设计
      • read.i: npm模块, 解决Node.js中异步标准I/O的麻烦,方便终端输入
      • colorlogger: npm模块, 作为RocketEngine库的工具包, 实现终端日志的彩色输出与保存
    • Ruby
      • gitdb: gem模块，封装了Git的底层操作，是Git-Contacts的源动力.
    • CSS & HTML
      • inkiron: 暑期实训项目写的类flat UI风格的前端CSS框架.
